# Game Ideas Roadmap

## High-Impact Next Moves

1. **Time-Trial Mode per Level**
   - Track best completion time by level.
   - Add quick-restart flow for repeated runs.
   - Add medal tiers (e.g., Bronze/Silver/Gold).

2. **Collectible Challenge Goals**
   - Add optional per-level objectives:
     - Collect all big coins.
     - Finish with no deaths.
     - Clear with a specific character.
   - Surface challenge completion in HUD or end-of-level summary.

3. **Checkpoint System for Long Levels**
   - Add lightweight checkpoints in larger stages.
   - Respawn from latest checkpoint instead of start.
   - Keep it optional/tunable per level.

4. **Character-Specific Skill Maps**
   - Build levels designed around each special skill.
   - Examples:
     - Grapple-focused map (Sky Ruins style).
     - Robot magnet gauntlet in Factory.
   - Improves character identity and replayability.

5. **Title/Menu Flow**
   - Add a simple front-end menu:
     - Start Game
     - Level Select
     - Character Select
     - Mute/Volume controls
   - Improves usability and first-time player experience.

## Polish Pass Ideas

1. **Enemy Variety Behaviors**
   - Add new AI patterns:
     - Patrol jumper
     - Ranged flyer
     - Shielded walker

2. **Feedback Effects Expansion**
   - Add more visual/audio feedback:
     - Landing dust
     - Hit flash
     - Perfect grapple spark
     - Combo popups

3. **Difficulty Curve Tuning**
   - Reorder and tune stages to teach mechanics progressively.
   - Ensure each level introduces one core idea, then tests it.

## Recommended First Build

**Start with: Time-Trial Mode + Medals**

Why this first:
- Adds immediate replay value.
- Low implementation risk compared to larger system changes.
- Gives players clear mastery goals without changing core controls.

## Suggested Implementation Order

1. Time-Trial + Medals
2. Collectible Challenge Goals
3. Checkpoints
4. Character-Specific Skill Maps
5. Title/Menu Flow
6. Enemy Variety + FX + Difficulty Tuning
